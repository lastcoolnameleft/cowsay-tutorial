# Prerequisite

Make sure to install cowsay first.  (Currently only tested on Ubuntu)

```shell
apt-get install cowsay
export PATH=/usr/games:$PATH
```

# Validation

Verify that you have cowsay installed correctly.

```
cowsay hi
```

Results:

```
 ____
< hi >
 ----
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
```
